Tools

A rich ecosystem of tools is growing around Builder.

Java SeInterpreter

Download, Docs
Se Interpreter is a command-line tool for playing back Selenium 2 .json files created by Builder. Instead of exporting scripts to a specific programming language for playback, you can keep them in their editable .json form and use Interpreter.

node se-interpreter

NPM & Docs
An interpreter for .json files built on node.js, with additional features for complex run configurations. Install using npm.

simple-selenium-builder-framework

GitHub Project
Selenium test framework designed to run Selenium Builder recorded tests on SauceLabs using TestNG as a threaded test runner.
